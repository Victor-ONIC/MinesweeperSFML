
Minesweeper.


VARIABLES:
- size of the window: width and height.
- dimensions of the grid: nb_row and nb_col
- number of mines


------
Start with the empty window.

Create the game matrix.
Display the grid. Each cell in the matrix draws one square.

(start loop) Wait for user input.

Dig the cell which the user clicked on.
(first time) Generate the mines.
Refresh the game matrix (grosse partie).
Refresh the display.
